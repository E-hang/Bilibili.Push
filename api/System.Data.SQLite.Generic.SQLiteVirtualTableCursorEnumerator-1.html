SQLiteVirtualTableCursorEnumerator<T> 类

This class represents a virtual table cursor to be used with the SQLiteModuleEnumerable class. It is not sealed and may be used as the base class for any user-defined virtual table cursor class that wraps an System.Collections.Generic.IEnumerator<T> object instance.
