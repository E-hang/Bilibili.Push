SQLiteDataReaderValue 类

This class represents a single value to be returned from the SQLiteDataReader class via its GetBlob(Int32, Boolean), GetBoolean(Int32), GetByte(Int32), GetBytes(Int32, Int64, Byte[], Int32, Int32), GetChar(Int32), GetChars(Int32, Int64, Char[], Int32, Int32), GetDateTime(Int32), GetDecimal(Int32), GetDouble(Int32), GetFloat(Int32), GetGuid(Int32), GetInt16(Int32), GetInt32(Int32), GetInt64(Int32), GetString(Int32), or GetValue(Int32) method. If the value of the associated public field of this class is null upon returning from the callback, the null value will only be used if the return type for the SQLiteDataReader method called is not a value type. If the value to be returned from the SQLiteDataReader method is unsuitable (e.g. null with a value type), an exception will be thrown.
