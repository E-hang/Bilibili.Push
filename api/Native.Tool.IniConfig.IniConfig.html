异常
类型 条件

System.Runtime.InteropServices.InvalidComObjectException

未通过 Overload:System.Type.GetTypeFromProgID(System.String) 或 Overload:System.Type.GetTypeFromCLSID(System.Guid) 获取 COM 类型
System.Runtime.InteropServices.COMException

t 是一个 COM 对象，但用于获取类型的类标识符无效，或标识的类未注册
System.TypeLoadException

t 不是有效类型
