SQLiteBackupCallback 委托

Raised between each backup step.

命名空间: System.Data.SQLite
程序集: Native.Tool.dll
语法
public delegate bool SQLiteBackupCallback(SQLiteConnection source, string sourceName, SQLiteConnection destination, string destinationName, int pages, int remainingPages, int totalPages, bool retry);
参数
类型 名称 描述
SQLiteConnection source

The source database connection.
System.String sourceName

The source database name.
SQLiteConnection destination

The destination database connection.
System.String destinationName

The destination database name.
System.Int32 pages

The number of pages copied with each step.
System.Int32 remainingPages

The number of pages remaining to be copied.
System.Int32 totalPages

The total number of pages in the source database.
System.Boolean retry

Set to true if the operation needs to be retried due to database locking issues; otherwise, set to false.
返回
类型 描述
System.Boolean

True to continue with the backup process or false to halt the backup process, rolling back any changes that have been made so far.
