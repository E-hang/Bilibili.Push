SQLiteModuleEnumerable<T> 类

This class implements a virtual table module that exposes an System.Collections.Generic.IEnumerable<T> object instance as a read-only virtual table. It is not sealed and may be used as the base class for any user-defined virtual table class that wraps an System.Collections.Generic.IEnumerable<T> object instance.
